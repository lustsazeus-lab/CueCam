# AndriodWordHint
安卓提词器

## 新增能力（2026-04）
- 提词页支持打开摄像头预览，并可一键切换前置/后置摄像头。
- 在提词页内可直接开始/停止视频录制，录制文件会保存在应用私有目录（Movies/Teleprompter）。
- 保留全屏提词体验：开启摄像头后提词层会自动改为半透明遮罩，文字仍可全屏滚动显示。
- 支持调用系统录像应用（`ACTION_VIDEO_CAPTURE`）。系统相机内的美颜/滤镜/高级设置由手机厂商相机 App 决定。
- 增加提词遮罩透明度滑杆、分辨率/FPS 选择、录制计时器（REC）。
- 提词控制面板改为“毛玻璃”风格（半透明渐变+高光描边）并统一按钮视觉。
- 稿件选择从弹窗升级为全屏“稿件页”，仅展示标题列表，便于快速筛选和触达。
